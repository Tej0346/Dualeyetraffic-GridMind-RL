# GridMind-RL: Advanced Traffic Signal Control
A Multi-Agent Reinforcement Learning environment for optimizing urban traffic flow.

## API Endpoints
- `/reset?task=easy`: Initialize environment
- `/step`: Execute action
- `/state`: Get current traffic state
- `/grader`: Get performance score
